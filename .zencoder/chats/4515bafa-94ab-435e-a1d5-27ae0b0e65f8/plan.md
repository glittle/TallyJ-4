# .zencoder Planning Artifacts

This directory is used for local AI-assisted planning and draft notes during development.

These files are **not** required for building, testing, or running TallyJ 4 and should not be
treated as canonical documentation or part of the supported project surface.

Repository documentation and implementation guidance should instead live in the maintained
project docs, such as:

- `README.md`
- `backend/SETUP.md`
- `frontend/README.md`
- `docs/DEPLOYMENT.md`
- `AGENTS.md`

If you see this file in a shared branch, it is safe to delete it and add the `.zencoder/`
directory to `.gitignore` so that future planning artifacts are not committed.
